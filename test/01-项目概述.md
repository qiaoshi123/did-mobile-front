# 项目概述

本文档定义 DID 项目的技术栈和开发环境。

## 技术栈

| 类别 | 技术 | 版本要求 |
|-----|------|---------|
| 前端框架 | Vue | 3.4+ |
| 脚本语言 | TypeScript | 4.9+ |
| 跨端框架 | UniApp | 3.0+ |
| 状态管理 | Pinia | 2.x |
| UI 组件库 | @tdesign/uniapp | 最新版 |
| 构建工具 | Vite | 5.x |
| 开发工具 | CodeBuddy IDE | - |

## 开发环境要求

- Node.js >= 18.0.0
- pnpm >= 8.0.0
- HBuilderX 或 CLI 开发

## 支持的平台

- H5（Web）
- 微信小程序
- 支付宝小程序
- App（iOS/Android）

## 项目特点

1. **DID（去中心化身份）**：支持区块链身份验证
2. **多端兼容**：一套代码同时支持小程序和 App
3. **AI Coding**：使用 SpecKit 规范驱动开发

## 代码质量目标

- TypeScript 严格模式
- 组件 100% 类型
- 自动化测试覆盖率覆盖 > 70%
