# 多端适配规范

本文档定义 UniApp 多端（App、小程序、H5）适配规范。

## 条件编译

### 语法说明

| 语法 | 说明 |
|-----|------|
| `// #ifdef xxx` | 如果满足条件，则编译 |
| `// #ifndef xxx` | 如果不满足条件，则编译 |
| `// #endif` | 结束条件 |

### 平台标识

| 标识 | 说明 |
|-----|------|
| `H5` | H5 端 |
| `MP-WEIXIN` | 微信小程序 |
| `MP-ALIPAY` | 支付宝小程序 |
| `APP-PLUS` | App 端 |
| `APP-PLUS-NVUE` | App NVue |
| `MP` | 所有小程序端 |

## 模板条件编译

```vue
<template>
  <!-- H5 端显示 -->
  <view v-if="isH5" class="header-h5">
    <text>H5 专属头部</text>
  </view>

  <!-- 小程序端显示 -->
  <view v-if="isMP" class="header-mp">
    <text>小程序专属头部</text>
  </view>

  <!-- App 端显示 -->
  <view class="header-app">
    <text>App 专属头部</text>
  </view>
</template>

<script setup lang="ts">
// #ifdef H5
const isH5 = true
// #endif

// #ifdef MP
const isMP = true
// #endif
</script>
```

## 样式条件编译

```scss
/* 通用样式 */
.container {
  padding: 24rpx;
}

/* H5 端样式 */
#ifdef H5
.container {
  padding-top: 44px; // 适配 H5 导航栏
}
#endif

/* 小程序端样式 */
#ifdef MP-WEIXIN
.container {
  padding-bottom: 100rpx; // 适配小程序安全区域
}
#endif

/* App 端样式 */
#ifdef APP-PLUS
.container {
  padding-bottom: calc(20rpx + constant(safe-area-inset-bottom));
  padding-bottom: calc(20rpx + env(safe-area-inset-bottom));
}
#endif
```

## API 条件编译

```typescript
import { platform } from '@/utils/platform'

// #ifdef H5
import { H5SpecificApi } from '@/utils/h5'
// #endif

// #ifdef MP-WEIXIN
import { WechatSpecificApi } from '@/utils/wechat'
// #endif

export function getPlatformInfo() {
  // #ifdef H5
  return H5SpecificApi.getInfo()
  // #endif

  // #ifdef MP-WEIXIN
  return WechatSpecificApi.getInfo()
  // #endif

  // #ifndef H5
  return uni.getSystemInfoSync()
  // #endif
}
```

## 常见差异处理

### 1. 导航栏

```scss
// pages.json
{
  "pages": [
    {
      "path": "pages/index/index",
      "style": {
        // H5 使用自定义导航
        // #ifdef H5
        "navigationStyle": "custom",
        // #endif
        "navigationBarTitleText": "首页"
      }
    }
  ]
}
```

### 2. 底部安全区域

```scss
.tabbar {
  // #ifndef H5
  padding-bottom: constant(safe-area-inset-bottom);
  padding-bottom: env(safe-area-inset-bottom);
  // #endif
}
```

### 3. 登录方式

```typescript
// 登录方式差异
async function login() {
  // #ifdef H5
  await h5Login()
  // #endif

  // #ifdef MP-WEIXIN
  await wechatLogin()
  // #endif

  // #ifdef APP-PLUS
  await appLogin()
  // #endif
}
```

### 4. 分享功能

```typescript
// #ifdef MP-WEIXIN
onShareAppMessage(() => {
  return {
    title: '分享标题',
    path: '/pages/index/index'
  }
})
// #endif
```

### 5. 位置信息

```typescript
function getLocation() {
  // #ifdef H5
  return new Promise((resolve) => {
    navigator.geolocation.getCurrentPosition((pos) => {
      resolve({
        latitude: pos.coords.latitude,
        longitude: pos.coords.longitude
      })
    })
  })
  // #endif

  // #ifndef H5
  return new Promise((resolve, reject) => {
    uni.getLocation({
      type: 'wgs84',
      success: resolve,
      fail: reject
    })
  })
  // #endif
}
```

## 资源路径差异

```typescript
// 动态获取资源路径
function getImagePath(name: string) {
  // #ifdef H5
  return `/static/images/${name}.png`

  // #ifndef H5
  return `https://cdn.example.com/images/${name}.png`
```

### 常见问题

| 问题 | 解决方案 |
|-----|---------|
| 样式不一致 | 使用条件编译 |
| API 不支持 | 使用条件导入 |
| 组件差异 | 使用多端组件 |
| 安全区域 | 使用 safe-area-inset |

## 禁止事项

1. **禁止忽略多端差异**
2. **禁止使用非通用 API**
3. **禁止不测试直接发布**
4. **禁止硬编码平台特定内容**
