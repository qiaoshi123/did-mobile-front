# 组件规范

本文档定义 Vue 3 组件的开发规范。

## 组件分类

### 1. 通用组件
- 位置：`src/components/`
- 特点：跨页面复用
- 示例：Button、Input、Loading、Empty

### 2. 业务组件
- 位置：`src/components/<business>/`
- 特点：特定业务域复用
- 示例：CredentialCard、UserAvatar

### 3. 页面级组件
- 位置：`src/pages/<page>/components/`
- 特点：仅当前页面使用

## 组件结构

```vue
<template>
  <view class="component-name">
    <!-- 组件内容 -->
  </view>
</template>

<script setup lang="ts">
import { ref, computed } from 'vue'
import type { Props, Emits } from './types'

// Props 定义
const props = withDefaults(defineProps<Props>(), {
  type: 'primary',
  disabled: false
})

// Emits 定义
const emit = defineEmits<Emits>()

// 响应式状态
const isLoading = ref(false)

// 计算属性
const classes = computed(() => [
  'component-name',
  `component-name--${props.type}`,
  { 'is-disabled': props.disabled }
])

// 方法
function handleClick() {
  if (props.disabled || isLoading.value) return
  emit('click')
}
</script>

<style scoped lang="scss">
@import '@/styles/variables.scss';

.component-name {
  // 样式
}
</style>
```

## Props 定义规范

```typescript
// types.ts
export interface Props {
  /** 按钮类型 */
  type?: 'primary' | 'secondary' | 'ghost'
  /** 是否禁用 */
  disabled?: boolean
  /** 加载状态 */
  loading?: boolean
  /** 点击回调 */
  onClick?: () => void
}
```

### Props 命名规则

| 规则 | 说明 |
|-----|------|
| 类型以 `?` 结尾 | 表示可选 |
| 必须有 JSDoc 注释 | 说明用途 |
| 布尔类型默认值 | 根据场景设置 |

## 组件通信

### Props 传递

```vue
<!-- 父组件 -->
<ChildComponent
  :user="userInfo"
  :loading="isLoading"
  @success="handleSuccess"
/>
```

### Emits 事件

```typescript
// 子组件
const emit = defineEmits<{
  (e: 'success', data: UserInfo): void
  (e: 'error', error: Error): void
  (e: 'update:visible', visible: boolean): void
}>()

// 触发
emit('success', data)
```

### Provide / Inject

```typescript
// 祖孙组件通信
// provider
import { provide } from 'vue'
import type { UserInfo } from '@/types'

provide('userInfo', userInfo)

// consumer
import { inject } from 'vue'
const userInfo = inject<UserInfo>('userInfo')
```

## 插槽规范

```vue
<!-- 默认插槽 -->
<slot />

<!-- 命名插槽 -->
<view class="header">
  <slot name="header" />
</view>

<!-- 作用域插槽 -->
<slot :item="item" :index="index" />
```

## 组件引入

### TDesign 组件（按需引入）

```typescript
// ✅ 推荐：按需引入
import { Button, Input, Cell } from '@tdesign/uniapp'

// ❌ 禁止：全量引入
// import TD from '@tdesign/uniapp'
```

### 自定义组件

```typescript
// 在 components/index.ts 统一导出
export { default as Loading } from './loading/index.vue'
export { default as Empty } from './empty/index.vue'
```

## 禁止事项

1. **禁止直接修改 props**
2. **禁止在组件内部直接使用 uni API**
3. **禁止内联样式（除动态样式）**
4. **禁止使用行内样式覆盖组件样式**
