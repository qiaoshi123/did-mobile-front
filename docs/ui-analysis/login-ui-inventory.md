### 关键 UI 要点
- 页面顶部有大面积品牌装饰图（占页面约 45%），使用 dummyimage 占位
- 登录表单无外边框/卡片容器，直接在白色背景上排列
- 输入框采用底部分隔线样式（非边框样式），左侧带蓝色图标
- 密码输入框右侧有密码显示/隐藏切换图标
- "创建新的企业账号"和"忘记密码？"两端对齐
- 登录按钮文字 "登 录" 中间有空格
- 页面无导航栏（全屏页面）
- 右上角有蓝色盾牌勾选装饰图标

### 可复用的 TDesign 组件
- `<t-input>` — 企业账号输入框、密码输入框
- `<t-button>` — 登录按钮
- `<t-icon>` — 用户图标、锁图标、密码显隐图标、盾牌图标
