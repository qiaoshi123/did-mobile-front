## 三、样式规范汇总

### 颜色映射

> **TDesign 变量优先**：当设计稿色值与 TDesign 变量功能定位相同但数值略有差异时，一律映射到 TDesign 变量，不追设计稿精确色值。备注中记录"设计稿原值 #xxx，采用 TDesign 变量"。

| 用途 | 设计稿色值 | TDesign 变量 | 备注 |
|------|-----------|-------------|------|
| 主文本色（输入内容） | #333333 | var(--td-text-color-primary) | |
| 次要文本（忘记密码） | #999999 | var(--td-text-color-secondary) | 设计稿偏灰，采用 TDesign 变量 |
| 占位文本 | #BBBBBB | var(--td-text-color-placeholder) | 设计稿原值 #BBB，采用 TDesign 变量 |
| 页面背景（表单区域） | #FFFFFF | var(--td-bg-color-container) | |
| 主题色/品牌色 | #0052D9 | var(--td-brand-color) | 登录按钮 + 链接文字 |
| 分隔线颜色 | #E6E6E6 | var(--td-border-level-1-color) | 输入框底部分隔线 |
| 按钮文字色 | #FFFFFF | var(--td-text-color-anti) | 白色按钮文字 |

### 字体规范
| 用途 | 字号(rpx) | 字重 | 行高 | 使用场景 |
|------|----------|------|------|---------|
| 输入框文字 | 30 | 400 | - | 账号/密码输入 |
| 输入框 placeholder | 30 | 400 | - | 占位提示 |

### 间距规范
| 层级 | 值(rpx) | 使用场景 |
|------|--------|---------|

| 输入项间垂直间距 | 40 | 账号输入 → 密码输入 |
| 输入项到快捷链接间距 | 32 | 密码输入 → 快捷链接行 |
| 快捷链接到按钮间距 | 60 | 快捷链接 → 登录按钮 |

| 输入框底部 padding | 16 | 输入内容 → 分隔线 |

| 元素类型 | 值(rpx) |
|----------|--------|
| 登录按钮 | 48（半圆角） |
| 输入框 | 无圆角（底部分隔线风格） |

## 七、给 create-proposal 的摘要

### 页面清单
- 页面名称：登录页
- 页面路径建议：`/pages/login/index`
- 页面类型：表单页

### 需要新建的组件
| 组件名 | 分类 | 简述 |

### 关键 UI 要点
- 顶部为大面积品牌装饰图（≈ 540rpx 高），使用占位图实现
- 表单采用底部分隔线风格而非边框包围风格（TDesign Input 的 borderless 模式 + 自定义底部 border）
- 输入框左侧有前置图标（用户图标、密码图标）
- 密码输入框右侧有密码可见性切换图标
- 登录按钮为全宽圆角样式（shape="round"）
- "创建新的企业账号" 和 "忘记密码？" 为两个快捷链接，横向两端对齐
- 页面无导航栏、无底部 TabBar
- 登录按钮文字 "登 录" 中间有空格
- 页面水平边距为 60rpx（略大于常规 32rpx）

### 可复用的 TDesign 组件
- `<t-input>` — 账号输入框 + 密码输入框
- `<t-button>` — 登录按钮
