# 交互

##  表单验证
- 企业账户名，3-10位，不能包含空格。 汉字、字母、数字、下划线、减号或其组合。只能以汉字或字母开头。
- 手机号/邮箱， 手机格式正则校验 ，邮箱格式正则校验
- 密码 6-16位，包含数字、字母。 密码可点击小眼睛控制切换显示明文还是密文。
- 二次确认密码 6-16位，包含数字、字母，与密码保持一致。 密码可点击小眼睛控制切换显示明文还是密文。
- 管理员名称，非空，3-10位，汉字。
- 验证码，非空。

## 协议勾选
- 单选，必须勾选。

## 下一步按钮
- 点击按钮后，检查协议勾选，进行表单验证。验证通过后，调用 API 进行注册。注册成功后，跳转到首页。
