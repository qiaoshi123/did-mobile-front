# 2026-03-23

## create-page Skill 修改：组件创建复用 create-component Skill

修改了 `create-page` Skill（`.codebuddy/skills/create-page/SKILL.md`），将第 2 步从"自行内联编写组件创建逻辑"改为"调用 `use_skill('create-component')` 复用 create-component Skill"。

具体改动 3 处：
1. **场景一第 2 步**：删除原来的内联逻辑（读 04-组件规范 Rule → 判断定位 → 创建文件 → 导出），替换为调用 create-component Skill 的 4 步流程（调用 Skill → 传入需求 → Skill 自治 → 继续下一个），并明确禁止内联编写
2. **场景二第 2 步**：补充说明如有待新建组件，同样调用 `use_skill("create-component")` 完成创建
3. **两个检查清单**：更新组件创建相关的检查项，明确通过 create-component Skill 创建

目的：组件创建只有一套标准（create-component Skill），避免两边逻辑漂移。
