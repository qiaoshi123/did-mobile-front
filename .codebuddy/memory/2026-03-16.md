## 新增 API 接口：私钥恢复

**服务**：did-app（`src/http/services/did-app/`）

**接口信息**：
- 名称：私钥恢复
- 方法：POST `/auth/privateKeyRecovery`
- 请求体：`{ priKey: string }` — 其中1/3份私钥(base64编码)
- 响应数据：`{ privateKey: string }` — 后端恢复的私钥(base64编码)

**变更文件**：
1. `api-types.ts` — 新增 `DidappPrivateKeyRecoveryBody`、`DidappPrivateKeyRecoveryResult` 类型
2. `index.ts` — 新增 `didappPrivateKeyRecovery` API 函数，放在私钥管理模块下
