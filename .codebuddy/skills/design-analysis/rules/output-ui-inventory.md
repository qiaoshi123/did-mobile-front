## 使用说明

1. **复制模板**：将上述模板复制到 `docs/ui-analysis/<名称>-ui-inventory.md`
2. **替换占位符**：将 `___`、`<名称>` 等占位符替换为实际内容
3. **填充内容**：根据前 4 步的分析结果，填充各章节
4. **检查完整性**：使用 `workflow-output-checklist.md` 中的质量检查清单验证
