# TDesign UniApp 组件库技能

## 触发条件

**在以下场景中自动调用此技能：**

1. 开发 uni-app 页面时使用 TDesign 组件库
2. 使用 `t-` 前缀的组件时（如 `t-button`, `t-input`, `t-form` 等）
3. 需要表单、按钮、弹窗、提示、导航等 UI 组件时
4. 需要自定义主题、样式覆盖、深色模式等全局配置时
5. 需要 TDesign UniApp 组件的 API 参考和使用示例时

## 使用示例

### 按钮组件
```vue
<template>
  <t-button theme="primary" @click="handleClick">主要按钮</t-button>
  <t-button theme="default">默认按钮</t-button>
  <t-button theme="danger" variant="outline">危险描边按钮</t-button>
  <t-button theme="primary" size="large" block>通栏按钮</t-button>
</template>
```

### 输入框组件
```vue
<template>
  <t-input label="标签文字" placeholder="请输入文字" />
  <t-input placeholder="请输入文字" clearable />
  <t-input label="密码" type="password" placeholder="请输入密码" />
</template>
```

### 对话框组件
```vue
<template>
  <t-button theme="primary" @click="visible = true">打开对话框</t-button>
  <t-dialog
    v-model:visible="visible"
    title="对话框标题"
    content="告知当前状态、信息和解决方法"
    @confirm="onConfirm"
  />
</template>

<script setup>
import { ref } from 'vue';
const visible = ref(false);
const onConfirm = () => { visible.value = false; };
</script>
```

### 表单组件
```vue
<template>
  <t-form :data="formData" :rules="rules" ref="formRef">
    <t-form-item label="用户名" name="username">
      <t-input v-model="formData.username" placeholder="请输入用户名" />
    </t-form-item>
    <t-form-item label="密码" name="password">
      <t-input v-model="formData.password" type="password" placeholder="请输入密码" />
    </t-form-item>
  </t-form>
</template>
```

## 组件引入方式

TDesign UniApp 支持按需引入，推荐使用 easycom 自动导入：

```json
// pages.json
{
  "easycom": {
    "custom": {
      "^t-(.*)": "@tdesign/uniapp/$1/$1.vue"
    }
  }
}
```

也可以手动引入：
```js
import TButton from '@tdesign/uniapp/button/button.vue';
import TInput from '@tdesign/uniapp/input/input.vue';
```
